# 概率论基础笔记

## 随机试验

- **三个特点**：可重复性、可预知性、不确定性。

## 样本空间与样本点

- **样本空间**：试验上所有可能的结果，记为$\Omega$。
- **样本点**：样本空间的每一个元素，记为$\omega$。

## 随机事件

- 定义：样本空间$\Omega$的子集。

## 事件间的包含与运算

### 1. 包含

- 符号：$A \subset B$
- 含义：$A$发生，则必导致$B$发生。
- 相等：$A = B \iff A \subset B$且$B \subset A$

### 2. 和（并）

- 符号：$A \cup B$（或$A + B$）
- 含义：$A$发生或者$B$发生（$A$与$B$至少发生一个）。

### 3. 积（交）

- 符号：$AB$（或$A \cap B$）
- 含义：$A$发生且$B$发生（$A$和$B$同时发生）。

### 4. 差

- 符号：$A - B$
- 含义：$A$发生但$B$不发生。
- 等价式：$A - B = A - AB$

### 5. 互不相容（互斥）

- 符号：$AB = \varnothing$
- 含义：$A$与$B$不能同时发生。

### 6. 对立事件

- 定义：$A$与$B$对立$\iff AB = \varnothing$且$A \cup B = \Omega$。
- 等价式：$A - B = A - AB = A \cdot \overline{B}$

## 运算定律

### 1. 交换律

- $A \cup B = B \cup A$
- $A \cap B = B \cap A$

### 2. 结合律

- $(A \cup B) \cup C = A \cup (B \cup C)$
- $(A \cap B) \cap C = A \cap (B \cap C)$

### 3. 分配律

- $(A \cap B) \cup C = (A \cup C) \cap (B \cup C)$
- $(A \cup B) \cap C = (A \cap C) \cup (B \cap C)$

### 4. 德摩根律

- $\overline{A \cup B} = \overline{A} \cap \overline{B}$
- $\overline{A \cap B} = \overline{A} \cup \overline{B}$

## 频率与概率

### 频率

- 定义：$A$发生的频率$ f_n(A) = \frac{n_A}{n} $（$n_A$为$A$发生的次数，$n$为试验总次数）。

### 概率

- 定义：当试验$n$充分大时，$f_n(A) \to P$，记为$P(A) = P$。
- **性质**：
  - $0 \leq P(A) \leq 1$
  - $P(\Omega) = 1$（必然事件概率为1）
  - $P(\varnothing) = 0$（不可能事件概率为0）

## 重要公式

- $P(A - B) = P(A) - P(A \cap B)$
- 若$A$与$B$互斥（$AB = \varnothing$），则$P(A \cup B) = P(A) + P(B)$
